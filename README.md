# Разработка RAG системы

Требуется разработать систему для поиска правовых статей из Конституции Российской Федерации

## Технологический стек

1. Модель для формирование контекстуальных эмбеддингов текстовых документов: `deepvk/USER-bge-m3`

2. LLM: `gpt-oss:20b`

3. Векторная БД: `FAISS`

4. Reranker: `qilowoq/bge-reranker-v2-m3-en-ru`

5. Фреймворк для связывания компонентов: `LangChain`

6. Фреймворк для разработки API: `FastAPI`

7. Web-интерфейс: `streamlit`
